4.4 lpp

  • 作用

  • 生成绘制LPP相图的数据

  • 例子

    • 发送数据

    • 接收数据

4.4.1 必选参数

  • elements
    • 类型：List[str]
    • 作用：设定的三种化学元素，有且只有三种
  • species_list
    • 类型：List[str]
    • 作用：凝聚态物质的化学式列表
    • 注意：
      • 绘制成功，必须有不少于三种不相似分子
      • 但是，如果这三种物质没有三相点，还是会绘制失败
  • gas_list
    • 类型：List[str]
    • 作用：给定x，y轴的气体，有且只有两个
      • 第一个：x轴气体
      • 第二个：y轴气体

4.4.2 可选参数

  • temperature

    • 类型：double
    • 作用：指定需要固定的温度数值
    • 默认值：298.15
    • 注意：
      • 需要和可选参数degree_unit同时被设定
    • 可选值：
      • degree_unit 为 ”K“： > 0
      • degree_unit 为 ”C“：> -273.15

  • degree_unit

    • 类型：str
    • 作用：指定固定的温度单位
    • 默认值：”K”
    • 可选值：”K”或”C”
      • “K”：Kelvin
      • “C”：Celsius, ℃

  • energy_unit

    • 类型：str
    • 作用：指定能量单位
    • 可选值：”J”或”cal”
    • 默认值：”J”

  • xy_range

    • 类型：[[x_min, x_max], [y_min, y_max]] 或 null

      • 其中x_min, x_max, y_min, y_max 类型为double

    • 作用：

      • 范围：指定X，Y轴值（Log pGas）取值范围
      • null：由程序自动生成取值范围

    • 默认值：null

  • more_export

    • 类型：boolean
    • 作用：是否需要生成更多相关信息
      • 反应方程式等

4.4.3 生成数据解说

这里只解释结果{“data”: […], …}中，data下的数据，其他内容参见返回结果格式

  • 例子：

  • xy_range

    • 实际使用的x，y轴取值范围

  • shared_lines

    • LPP相图中的线，有多条

    • 类型：List[Dict]

      • name
        • 给定直线两边的物质化学式
        • 类型：List[str]
        • 有且只有两个元素
      • data
        • 给定直线的两个端点
        • 格式：[[x0, y0], [x1, y1]]
        • 类型：这里x，y都是double型

  • shared_triples

    • 作用：给定三相点的信息

    • 类型：List[Dict]

      • name
        • 相关的三个相的化学式
        • 类型：List[str]
        • 格式：[“phaseA”, “phaseB”, “phaseC”]
      • data
        • 三相点的位置
        • 类型：List[double]
        • 格式：[x, y]
          • x, y都是double

  • used_conves

    • 作用：给定相图中出现的凸包Convexes

      • 凸包：凸多边形

    • 类型：List[Dict]

      • name
        • 凸包对应的相的化学式
        • 类型：str
      • data
        • 凸包的顶点，逆时针排序
        • 类型：List[List[double]]

  • phases

    • 作用：相的化学式固定信息

    • 类型：List[Dict]

      • name
        • 相的化学式
        • 类型：str
      • data
        • 相固定的x，y坐标
        • 格式：[x, y]
          • x,y都是double

  • x

    • x轴的标签

  • y

    • y轴的标签

  • T

    • 固定温度数值

  • degree_unit

    • 固定温度单位

  • energy_unit

    • 计算使用能量单位

4.5 tpp

  • 作用：制作TPP相图绘图数据

  • 例子

    • 发送数据

    • 接收数据

4.5.1 必选参数

  • elements

    • 作用：指定选择的化学元素，有且仅有三种
    • 类型：List[str]

  • species_list

    • 作用：选定的物质化学式列表
    • 类型：List[str]
    • 注意：
      • 至少选定两种非相似化学式

  • variable_gas

    • 作用：指定分压变化气体化学式
    • 类型：str

  • fix_gas

    • 作用：指定分压固定气体化学式
    • 类型：str

  • fix_pressure

    • 作用：指定分压固定气体的固定压强值，单位bar
    • 类型：double

  • temp_limit

    • 作用：温度坐标轴的数值变化范围
    • 格式：[temp_min, temp_max]
    • 类型：temp_min, temp_max都是double
    • 注：
      • 与可选参数degree_unit一起设定
      • 注意温度的取值范围
        • K：>0
        • ℃：> -273.15

  • logp_limit

    • 作用：分压对数坐标轴的数值变化范围
    • 格式：[logp_min, logp_max]
    • 类型：logp_min, logp_max都是double
    • 注：
      • 量纲一的量，不需要指定单位

  • pixel_count

    • 作用：程序内部使用像素图计算的解析度
      • 注：
        • 数值大，精确度告
        • 数值小，计算速度快
    • 类型：int
      • 注意：这里是整数

4.5.2 可选参数

  • degree_unit
    • 作用：指定温度单位
    • 类型：str
    • 备选值：”K”或”℃”
    • 默认值：”K”
  • temp_axis
    • 作用：指定温度使用的坐标轴，x轴还是y轴
    • 类型：str
    • 备选值：”x”或“y”
    • 默认值：“x”

4.5.3 生成数据解说

这里只解释结果{“data”: […], …}中，data下的数据，其他内容参见返回结果格式

  • 例子

  • phases

    • 作用：相的标签固定信息

    • 类型：List[Dict]

      • name
        • 相的化学式
        • 类型：str
      • data
        • 固定位置的x，y坐标
        • 格式：[x, y]
          • x, y都是double类型

  • shared_lines

    • 作用：相之间的边界线信息

    • 类型：List[Dict]

      • name
        • 边界线两边的两个相的化学式
        • 类型：List[str]
        • 格式：[“phaseA”, “phaseB”]
      • data
        • 组成边界线的所有点
        • 格式：[[x0, y0], [x1, y1], …]
        • 类型：List[List[double]]

  • shared_triples

    • 作用

      • 说明三相点的信息
      • 注意，不一定必然有值，可能为空

    • 类型：List[Dict]

      • name
        • 给定三相点的三个相的化学式
        • 类型：List[str]
      • data
        • 给定三相点的坐标值
        • 格式：[x, y]
          • 其中x、y都是double类型

  • xy_range

    • 作用
      • 制作图像的x、y坐标轴取值范围
      • 注：和调用api时输入的xy_range一致
    • 格式：[[x_min, x_max], [y_min, y_max]]
      • 这里x_min、x_max、y_min、y_max都是double类型

  • x

    • 作用：说明x轴的标签
    • 类型：str

  • y

    • 作用：说明y轴的标签
    • 类型：str

  • degree_unit

    • 作用：说明温度的单位

    • 类型：str
